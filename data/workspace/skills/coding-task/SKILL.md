---
name: coding-task
description: Inspect, edit, and verify code changes in the current workspace.
---

Use this when the owner asks for code changes.

Steps:

1. Inspect files before editing.
2. Prefer narrow edits.
3. Run available checks: typecheck, tests, lint.
4. Do not claim success if checks fail.
5. Keep final report short: changed files, checks, blocker if any.
